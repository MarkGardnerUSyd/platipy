IO¶

platipy.imaging.utils.io.write_nrrd_structure_set(masks, output_file='structure_set.nrrd', color_map=<matplotlib.colors.LinearSegmentedColormap object>)¶

Write an nrrd structure given a set of masks. Useful for reading in Slicer.

Parameters

  • masks (dict) – Key is structure name, value is SimpleITK.Image

  • output_file (str, optional) – The path to write the nrrd file. Defaults to “structure_set.nrrd”.

  • color_map (matplotlib.colors.Colormap | dict, optional) – Colormap to use for output. Can also use a dictionary with keys as structure names and values as colors. Defaults to cm.get_cmap(“rainbow”).

Raises

  • AttributeError – masks must be a dict

  • ValueError – color_map must be a matplotlib colormap or a dictionary
